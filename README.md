# 🎬 MovieMetric — Explore & Track Movies

A modern, responsive movie discovery application built with **React** and powered by the **TMDB API**. This project handles real-time data fetching, search debouncing, and persistent data storage using **Appwrite**.

## ✨ Features

- **Dynamic Movie Discovery:** Fetches the latest and most popular movies directly from the TMDB database.
- **Advanced Search:** Real-time search functionality with **Custom Debouncing** to optimize API performance and reduce unnecessary network traffic.
- **Responsive Design:** A sleek, dark-mode UI fully optimized for mobile, tablet, and desktop views using **Tailwind CSS**.
- **Robust Error Handling:** Comprehensive `try/catch` implementation to handle API failures and loading states gracefully.

## 🛠️ Built With

- **Frontend:** [React.js](https://reactjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
